# Image Assets

Add your images to this folder with the following names:

## Profile Image
- `profile.png` - Your profile/avatar photo (recommended: 400x400px)

## Project Thumbnails
- `project-miaw-cv.png` - Miaw CV project screenshot
- `project-django-api.png` - Django REST API project screenshot
- `project-cv-builder.png` - CV Builder feature screenshot
- `project-portfolio.png` - Portfolio website screenshot
- `project-unity-game.png` - Unity game project screenshot
- `project-blender.png` - Blender 3D assets screenshot

## Recommended Sizes
- Profile: 400x400px (square)
- Project thumbnails: 800x600px (4:3 ratio)

## Supported Formats
- PNG (recommended for transparency)
- JPG/JPEG
- WebP
